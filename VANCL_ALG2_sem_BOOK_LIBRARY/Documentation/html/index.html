Aplikace pro objednávání knížek

Introduction

Do aplikace má přístup pouze uživatelé, kteří jsou registrováni v databázi. Aplikace umožňuje přihlášení manuální, nebo uložení posledního přihlášení. Aplikace předvyplní data, uživatel poze potvrdí přihlášení.

Menu aplikace se vytvoří na základě oprávnění uživatele.

User

Uživatele může listovat v databázi knížek na základě nastaveného filtru. Knížky přidané do košíku muže seřadit podle jednotlivých parametrů, nebo odstranit z košíku.

V záložce s objednávkou pouze dovyplní data a odešle obejdnávku. Book tab zapíše knížky do zapůjčení a nastaví datum vrácení.

SuperUser

  • Může mazat uživatele.
  • Může upravovat oprávnění jednotlivých uživatelů.
